# 🚗 AutoFleet AI - Smart Fleet Management System

Un dashboard full-stack pentru managementul flotelor auto, care integrează diagnosticare predictivă AI. 

## 🛠️ Tech Stack
* **Backend:** Java 21, Spring Boot 4.x, Spring Data JPA
* **Frontend:** React, Vite, Tailwind CSS
* **Database:** MySQL 8.0
* **Infrastructure:** Docker & Docker Compose

## 🚀 Cum rulezi proiectul (prin Docker)
1. Asigură-te că ai Docker Desktop pornit.
2. Rulează comanda în folderul principal: `docker-compose up -d --build`
3. Frontend: `http://localhost:5173`
4. Backend API: `http://localhost:8080`
5. Baza de date UI (DbGate): `http://localhost:8081`
